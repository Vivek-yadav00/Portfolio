## 🛠️ Technologies Used

This portfolio is built using core web technologies to ensure maximum performance, quick load times, and fine-grained control over aesthetics:

- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom Vanilla CSS, Flexbox, CSS Grid, animations, and glassmorphism effects.
- **JavaScript (ES6+):** DOM manipulation, modal handling, project filtering, and smooth scrolling.
- **FontAwesome:** Scalable vector icons.
- **Google Fonts:** Inter and JetBrains Mono for clean, modern typography.
